{% extends 'lxd/base.html' %} {% block content %} Access the containers by name on the host
Taken from archlinux.org (https://wiki.archlinux.org/title/LXD#Access_the_containers_by_name_on_the_host)

Create a /etc/systemd/network/lxd.network that contains (replace x and y to match your bridge IP): 
    [Match]
    Name=lxdbr0
    [Network]
    DNS=10.x.y.1
    Domains=~lxd
    IgnoreCarrierLoss=yes
    [Address]
    Address=10.x.y.1/24
    Gateway=10.x.y.1
And then enable and start systemd-networkd.service.
